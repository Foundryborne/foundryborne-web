System Features

Foundryborne is designed to provide a comprehensive and immersive Daggerheart experience on FoundryVTT.

Core Mechanics

  • Character Creation: Complete character creation wizard.
    • Support for Mixed Ancestries.
    • Support for Homebrew content throughout all options.
  • Dice Mechanics: Full support for Hope and Fear dice mechanics.
  • Fear Tracker: Built-in tracker for GM Fear.
  • Dice So Nice!: Support for Dice So Nice! is built into the system to allow customization of the hope & fear, as well as advantage & disadvantage dice.
  • Measuring System: Support for distance ranges, from Melee, Very Close, Close, Far and very Far.
  • Environments: Support for environments.

Automation & Tools

  • Optional Automation: Streamline your game with optional automation settings.
  • Encounter Tracker: Manage combat and encounters efficiently.
    • Players can easily request the spotlight from the GM and optionally have 3 action tokens each.
    • The GM can see who requested the spotlight (in order) and can grant it to them.
  • Countdown Tracker: Optional tracker for encounter and project countdowns.

Party Management

  • Party Actor: A dedicated actor for managing the party.
    • Group Rolls: Easily handle group checks and rolls.
    • Tag Team Rolls: Support for tag team mechanics.
    • Party Inventory: Manage the party's inventory.
    • Party Overview: A quick overview of the party's stats.

Customization & Homebrew

Extensive support for homebrew content, allowing you to create custom:

  • Domains, Domain Cards, Classes, and Subclasses
  • Ancestries and Communities
  • Features and Items
  • Adversary Types, Adversaries, and Environments
