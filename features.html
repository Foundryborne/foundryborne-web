System Features

Foundryborne is designed to provide a comprehensive and immersive Daggerheart experience on FoundryVTT.

SRD Content

Includes the full SRD contents, which covers all adversaries, items, environments, domain cards, ancestries, classes, subclasses and other character options from the core rulebook, as well as the Witherwild campaign frame.

Core Mechanics

  • Character Creation: Complete character creation wizard.
    • Support for Mixed Ancestries.
    • Support for Homebrew content throughout all options.
  • Dice Mechanics: Full support for Hope and Fear dice mechanics.
  • Fear Tracker: Built-in tracker for GM Fear.
  • Dice So Nice!: Support for Dice So Nice! is built into the system to allow customization of the hope & fear, as well as advantage & disadvantage dice.
  • Measuring System: Support for distance ranges, from Melee, Very Close, Close, Far and Very Far.
  • Death & Fate Rolls: Fully implemented Death Rolls with automation for death moves, and support for Fate Rolls.
  • Environments: Support for environments, and linking scene environments for quick access.

Automation & Tools

  • Optional Automation: Streamline your game with optional automation settings handling things like condition application as stress changes.
  • Encounter Tracker: Manage combat and encounters efficiently.
    • Players can easily request the spotlight from the GM and optionally have 3 action tokens each.
    • The GM can see who requested the spotlight (in order) and can grant it to them.
  • Adversary Management: Support for multi-phase Adversaries via Transform actions, scaling tiers, and preset token sizes.
  • Trigger System: Powerful system allowing actions to register macros that run at specified occurrences.
  • Countdown Tracker: Optional tracker for encounter and project countdowns.

Party Management

  • Party Actor: A dedicated actor for managing the party.
    • Group Rolls: Easily handle group checks and rolls.
    • Tag Team Rolls: Support for tag team mechanics.
    • Party Inventory: Manage the party's inventory and seamlessly drag and drop currency or items between characters.
    • Party Overview: A quick overview of the party's stats.
  • Companions: Support for companions including levelup choices and token clowncar behavior.

Customization & Homebrew

Extensive support for homebrew content, allowing you to create custom:

  • Domains, Classes, Subclasses, and advanced Domain Cards (Domain Touched, Vault Active, Loadout Ignore).
  • Ancestries and Communities.
  • Features and Items.
  • Adversary Types, Adversaries, and Environments.
  • Custom Resources to define your own tracked values.
  • Advanced Active Effects for damage multipliers, roll toggle settings, and more.

Class Specific Features

Features like Companion Sheets (Ranger), Beastform Transformations (Druid), Enhanced Automations like Strange Patterns (Wizard) and many more!
